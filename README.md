# 乐轮滑教练

基于Flutter写的一个教练打卡App,本项目已经上线使用，可在各大应用商店查看，为了大家学习和尝试Flutter项目，本着开源给大学一个例子供学习，仅供参考！

## 简单说明一下
本项目是公司正在运营的项目，为了安全起见，后端接口暂不开源，希望大家理解，App项目源码都在这里，可以对照线上App查看源码！
